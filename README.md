super-bee
=========

Game about bee, collect nectar from flowers ))

### Controls:

  Movement: A,S,D,W <br/>
  Shot: Mouse Left <br/>
  Reload: R <br/>
  Collect nectar: Space <br/>
  Bomb: Left Ctrl <br/>
  Pause: P <br/>
  Sound: Backspace <br/>
  Exit: Esc <br/>
